# lua-console

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

# Usage

CTRL + Enter to fire the code to the [back-end](https://github.com/piboistudios/MegameterConsole).
